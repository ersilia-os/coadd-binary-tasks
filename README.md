# CoADD and Spark binary tasks
Binary tasks from the Co-ADD antimicrobial drug discovery resource, including Spark

## Background

This repository creates binary tasks from CoADD and Spark datasets, available through the [CoADD website](https://co-add.org). Both datasets are aimed at antimicrobial drug discovery research.

## Steps

### Download the data

1. We manually downloaded the full dataset from CoADD. The download link is available [here](https://db.co-add.org/downloads/).
2. In addition, we downloaded Spark data from the CDD Vault. The download link is available [here](https://www.collaborativedrug.com/spark-data-downloads).
